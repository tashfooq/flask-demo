# Appointment Scheduling API

## Set Up

### Clone the Repo

> `git clone https://github.com/tashfooq/flask-demo.git`

### Pull the Docker Image

> `docker pull tashfooq/maven-appointment:latest`

### Navigate to Repo
> `cd ~/flask-demo`

## How To Use

### GET Request

***The URL***

> http://127.0.0.1:5000/api/appointments/<user_id>
